## Getting Started

### Prerequisites

No special prerequisites needed! Just a modern web browser.

### Installation

1. Download or clone all the files to your computer
2. Ensure you have these files in the same folder:
   - `index.html`
   - `styles.css`
   - `script.js`

### Running the Website

Simply open `index.html` in any modern web browser (Chrome, Firefox, Safari, Edge).

**Option 1: Double-click**
- Double-click on `index.html` to open it in your default browser

**Option 2: Right-click**
- Right-click on `index.html`
- Select "Open with" and choose your preferred browser

**Option 3: Local Server (Recommended for Development)**
- If you have Python installed, you can run:
  ```bash
  python -m http.server 8000
  ```
- Then open `http://localhost:8000` in your browser

## Customization

### Adding Contact Information

To add phone number and email:

1. Open `index.html`
2. Find the contact section (around line 200)
3. Replace the placeholder text with your actual contact details:
   ```html
   <p>+91 XXXXXXXXXX</p>
   <p>your-email@example.com</p>
   ```

### Modifying Colors

To change the color scheme:

1. Open `styles.css`
2. Find the `:root` section at the top
3. Modify the color variables:
   ```css
   --primary-color: #1e40af;  /* Main brand color */
   --secondary-color: #f59e0b; /* Accent color */
   ```

### Adding Images

To add vehicle images:

1. Create an `images` folder in the same directory
2. Add your images
3. Update the `.fleet-image` classes in `styles.css` to use background images instead of emojis
